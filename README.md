## サイト概要
### サイトテーマ
<映画の感想を語り合ったりいいね機能などで複数の人で評価するレビューサイト>

### テーマを選んだ理由
<私自身が映画が好きなので映画の評価を自分で発信したり、それを閲覧したりなどができるサイトを作りたいと思いました。>

<映画が好きな人、面白い映画を見つけたい人など>

### 主な利用シーン
<自分が見た映画の感想を発信したいとき、
 面白い映画を見つけたいとき、
 自分がこれから見ようとする映画の評判を見たいとき>
